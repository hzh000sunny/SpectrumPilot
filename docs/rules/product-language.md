# Product Language Rule

## Decision

SpectrumPilot is an English-first product.

The following names are finalized for the project foundation:

| Item | Value |
|---|---|
| GitHub repository name | `SpectrumPilot` |
| Executable name | `SpectrumPilot.exe` |
| Product name | `SpectrumPilot` |
| Product subtitle | `Wireless Research Assistant` |

## Scope

English should be used for:

- Application navigation.
- Page titles and descriptions.
- Buttons, forms, tables, empty states, settings, and notifications.
- Installer name and Windows Start Menu entry.
- Log messages that may be shown to end users.
- Public-facing documentation.

## Rationale

The product targets international wireless research workflows. The name should clearly signal the wireless domain while remaining broad enough to support future features beyond 3GPP downloading, including evidence search, patent disclosure drafting, PPT generation, and LLM-assisted analysis.

`SpectrumPilot` provides a wireless-domain signal through `Spectrum` and an assistant/workbench signal through `Pilot`.
